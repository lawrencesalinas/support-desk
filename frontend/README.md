# Support-Desk

Create a ticket for support. Support Desk is a MERN stack project(MongoDb, Express.js, React.js, Node.js),  CRUD application that utilizes the node.js environment, Express.js to handle the server, React.js with Redux for the frontend and MongoDb for saving data.
